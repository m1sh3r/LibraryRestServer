# LibraryRestServer

запуск ```go run main.go```

слушает порт 8000

Методы:

1. [GET]/cars - список всех автомобилей и информация по ним
2. [GET]/cars/{id} - информация по автомобилю с указанным id
3. [POST] /cars - добавление информации по новому автомобилю
4. [PUT] /cars/{id} - корректировка информации по автомобилю с указанным id
5. [DELETE]/cars/{id} - удаление информации по автомобилю с указанным id

Работает без БД, поэтому после запуска есть информация только по двум автомобилям: Ford Focus и Mitsubishi Lancer (хардкод).
